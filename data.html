Data

All data can be found in the data directory in our GitHub repo. All scraping scripts can be found under the scripts directory in the same repo. We got our data from the following sources:

  • NFL Play-by-Play Data: Spreadsheet Sports
  • Passer Rating + Rusher YPC/YPG: Pro Football Reference
  • Defense Rankings: Pro Football Reference
  • Weather: Weather Underground
